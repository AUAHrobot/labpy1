MENENTUKAN NILAI TERBESAR DARI 3 BILANGAN BILANGAN PERTAMA,KEDUA DAN KETIGA

1. Masukan Program di bawah ini pada PYCHARM

2. RUN Program tersebut

3. Berikut adalah hasilnya

4. Dibawah ini penjelasan program di atas

- Gunakan Syntax "print" untuk menampilkan kata pembuka.
- inisialisasi " a,b,c " sebagai integer dengan syntax "int dan input" untuk memasukan nilai bilangan
- Masukan Syntax "if" jika a>b dan a>c maka tampilkan " bilangan terbesar = a" dengan Syntax "print"
- Masukan Syntax “elif” jika b>c & b>c maka tampilkan “bilangan terbesar =b” dengan Syntax “print”
- Masukan Syntax “else” jika semua kondisi tidak tercapai
- Untuk menjalankan program klik "run"
- Masukan nilai bilangan pertama,kedua dan ketiga
- Selesai
